0. Prepare

   首先确保你有mysql环境，然后安装pymysql库

```
pip install pymysql
```

1. 打开sql_handler/config.py，将其中的mysql_server_password改成你的mysql登录密码

## Ubuntu运行

```
bash run.sh
```
## Windows 11运行

```
python gui/main.py
```
